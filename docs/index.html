This site develops SuSiE-relax, a fine-mapping method that lets the active LD column relax around a reference LD matrix via an inverse-Wishart prior, learning the relaxation level \(\nu_0\). The analyses below follow the progression of the method:

  1. SER-relax-prototype Demonstration - A prototype derived by making one approximation from susie-love

  2. SER-relax Demonstration - Making two more simple and benign approximations, we have a SER algorithm that is much faster and can still recover the true \(\nu_0\) and is interpretable.

  3. SER-collapse-R Demonstration - Direct collapsed-\(R\) single-effect fitting, comparing local \(q_j(\omega)\), shared \(q(\omega)\), and plug-in \(\omega=\nu_0+3\) variants.

  4. SuSiE-collapse-R Shared-Omega Analysis - The SuSiE-relax demonstration repeated with the collapsed-\(R\) shared-\(q(\omega)\) fitter.

  5. SuSiE-relax Demonstration - SuSiE extension of the SER model above.

  6. SuSiE-relax-prime Demonstration - The same SuSiE demonstration using the prime latent contribution model.

  7. Choosing lambda (low-mismatch case) - How the ridge parameter \(\lambda\) in \(\bar R = (1-\lambda) R_0 + \lambda I\) affects the fine-mapping result, using the final ELBO to select \(\lambda\).

  8. Choosing lambda (high-mismatch case) - The same \(\lambda\) investigation in the high-mismatch regime, where the reference panel is small and susieR on \(R_0\) overfits.

  9. Choosing lambda (different-ancestry case) - The same \(\lambda\) investigation when the GWAS and reference panels are simulated from different ancestries.

  10. PIP Comparison: SuSiE-relax vs Collapsed-R - Diagnostics for whether collapsed-\(R\) spreads PIP mass too diffusely in the different-ancestry setting.

  11. Scale of q as lambda varies - How \(q(\lambda)=x^\top \bar R(\lambda)^{-1}x\) changes with \(\lambda\) in low-mismatch, small-reference high-mismatch, and different-ancestry settings.
